# Data-Analytics-Science-Internship-Tasks
This repository contains tasks completed for Data Science &amp; Analytics Internship including data analysis, visualization, and machine learning models.

## Overview

This repository contains the tasks completed as part of the Data Science & Analytics Internship. The objective is to apply data analysis, visualization, and machine learning techniques to real-world datasets.

---

## Task 1: Iris Dataset Analysis

**Objective:**
To explore and visualize the dataset to understand feature relationships.

**Approach:**
Performed data exploration and used scatter plots, histograms, and box plots to analyze feature distributions and relationships.

**Results and Insights:**
Petal length and width were found to be the most important features for distinguishing species.

---

## Task 3: Customer Churn Prediction

**Objective:**
To predict whether a customer will leave the bank.

**Approach:**
Cleaned the dataset, encoded categorical variables, and trained a classification model (Random Forest).

**Results and Insights:**
Age, balance, and geography were identified as key factors influencing customer churn.

---

## Task 4: Insurance Claim Prediction

**Objective:**
To predict medical insurance charges based on personal data.

**Approach:**
Applied Linear Regression after encoding categorical variables and analyzing feature relationships.

**Results and Insights:**
Smoking, age, and BMI were found to significantly impact insurance costs.
